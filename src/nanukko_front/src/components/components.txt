컴포넌트(Component):
Vue 애플리케이션의 UI를 정의하는 기본 단위입니다. 템플릿, 스타일, 로직을 포함하여 화면에 그려지는 부분을 관리합니다.
데이터 바인딩, 이벤트 처리, 렌더링을 포함하여 실제 화면에 요소를 표시합니다.
컴포넌트는 UI와 관련된 로직과 렌더링 로직을 함께 다룹니다.
컴포넌트는 훅을 사용하여 필요한 로직을 가져와서 사용합니다.

컴포넌트에 들어갈 함수:
UI 관련 로직: 특정 컴포넌트에서만 사용하는 함수는 해당 컴포넌트 안에 작성합니다.
예: 버튼 클릭 핸들러, 특정 UI 요소를 관리하는 로직 등.
렌더링 로직: 컴포넌트의 템플릿과 직접적으로 관련된 함수는 컴포넌트 안에 작성합니다.
