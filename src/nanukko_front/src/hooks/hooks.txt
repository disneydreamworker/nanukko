생명주기 훅인듯?

훅(Hook):
Vue 3 Composition API의 기능을 활용하여 재사용 가능한 로직을 캡슐화하는 함수입니다.
상태 관리, 데이터 가져오기, 라우터 관리와 같은 공통 로직을 분리하여 여러 컴포넌트에서 쉽게 재사용할 수 있습니다.
훅은 UI와는 독립적이며, 로직만을 포함합니다. 따라서 렌더링을 포함하지 않습니다.

예제
useFetch.js: 일반적인 API 요청 로직을 관리하는 훅.
useFormValidation.js: 폼 검증 로직을 관리하는 훅.
useAuth.js: 인증 상태를 관리하고 관련 기능 (예: 로그인, 로그아웃)을 제공하는 훅.
useModal.js: 모달 창의 상태와 열기/닫기 기능을 관리하는 훅.
useWindowSize.js: 창 크기를 추적하고 반응하는 훅.

훅에 들어갈 함수:
재사용 가능한 로직: API 요청, 데이터 처리, 상태 관리 등 여러 컴포넌트에서 공통으로 사용하는 함수는 훅에 작성합니다.
예: 데이터 가져오기 함수, 라우터 관련 로직, 이벤트 처리 로직 등.
